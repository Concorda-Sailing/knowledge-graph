# Examples

This directory is intentionally minimal — worked examples that reference real endpoints are project data and belong in your project's `<data_dir>/examples/` directory, not in the framework.

For the JSON node and dossier shapes, see:

- **Node JSON**: `schema/node.schema.json` (this repo) and `schema/dependents.schema.json`
- **Dossier markdown**: `schema/dossier.schema.md`

For a hands-on look, see your project's `<data_dir>/examples/` directory if it has one.
